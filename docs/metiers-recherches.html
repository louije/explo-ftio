Prérequis — Cette API est dans le royaume PE Connect (OAuth individu). L'usager doit s'être identifié via le bouton « Se connecter avec France Travail ». Scopes requis : api_peconnect-metiersrecherchesv1 et cpmetiers.
L'API retourne au maximum 2 métiers recherchés issus du profil de compétences de l'usager. Tous les champs sauf rome, appellation et typologieEmploi sont facultatifs — leur présence dépend de ce que l'usager a renseigné dans son profil.
