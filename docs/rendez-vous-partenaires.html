Prérequis : jeton usager — Tous les endpoints de cette API exigent le header ft-jeton-usager, obtenu via l'API Rechercher usager. Le jeton est volatile (~1 journée). Les trois headers d'identification agent (pa-nom-agent, pa-prenom-agent, pa-identifiant-agent) sont également requis sur chaque appel.

📖 Lecture — Le même RDV relu

GET /rendez-vous-usager — RendezVousUsager

Lecture En relisant l'agenda de Karine, le même rendez-vous apparaît mais sous le schéma RendezVousUsager. Les commentaires dans le JSON montrent les transformations :

id → idOrigine — le même UUID, renommé
PARTENAIRE → CONSEILLER — l'initiateur est normalisé
Mme → MME — la civilité est mise en majuscules
champs plats → objet AdresseRDVU — l'adresse est restructurée
information → informationUtile — le champ est renommé
+ estEnConflit, historiqueEvenementList — ajoutés par France Travail
