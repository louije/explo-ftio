RIASEC — Le modèle RIASEC (Réaliste, Investigateur, Artistique, Social, Entreprenant, Conventionnel) permet de caractériser le profil d'intérêt de chaque compétence. Les champs riasecMajeur et riasecMineur utilisent les codes R I A S E C.
Authentification — API publique, royaume /partenaire (client credentials). Scopes requis : nomenclatureRome + api_rome-competencesv1. Aucun jeton usager nécessaire.
