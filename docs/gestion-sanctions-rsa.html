Prérequis — Cette API est dans le royaume /agent (client credentials). Chaque appel nécessite le header ft-jeton-usager obtenu via l'API Rechercher usager, ainsi que les trois headers d'identification agent : pa-nom-agent, pa-prenom-agent, pa-identifiant-agent.
Flux de traitement — France Travail propose une sanction (PROP_SANCTION) → le Conseil départemental saisit la décision (SAISIE_SANCTION) → puis valide (VALID_SANCTION) ou refuse (REFUS_SANCTION). La suspension peut être levée après remobilisation (VALID_LEVEE_SUSP).

F01 — Absence RDV L'allocataire a manqué un entretien de suivi le 15 janvier 2026. Un avertissement a été émis le 20 janvier.

⚖ Sanction proposée → Suspension RSA 50% pendant 1 mois (proposée par FT)
⚖ Décision validée → Suspension RSA 50% pendant 1 mois (validée par le CD)
📅 Application → du 1er février au 28 février 2026
Headers requis : jeton usager + identification agent (CD75-0234)
📅 Traitement

Chronologie de la sanction

22/01 — PROP_SANCTION France Travail propose la suspension de 50%
28/01 — SAISIE_SANCTION Le Conseil départemental saisit la décision
30/01 — VALID_SANCTION Sanction validée après examen du dossier

La suspension de 50% du RSA s'applique du 1er au 28 février 2026. L'allocataire peut être remobilisé pour obtenir une levée anticipée (VALID_LEVEE_SUSP).
