Accès public — Cette API est accessible en client_credentials (OAuth2, royaume /partenaire). Aucun jeton usager requis. Scopes : o2dsoffre et api_offresdemploiv2. La pagination est limitée à 150 résultats par requête (max index 3149).
Tri et pagination — Le paramètre range (ex. 0-49) pagine les résultats. Le header Content-Range indique la position dans le jeu total. Trois modes de tri : 0 pertinence · 1 date · 2 distance.
Concepts partagés — Voir les concepts croisés entre APIs dans le catalogue
Cette API EST un service de référentiels — En plus de la recherche d'offres, elle expose 15 endpoints GET /v2/referentiel/… qui servent les catalogues utilisés à la fois par le moteur d'offres et, plus largement, comme source de vérité pour d'autres APIs du réseau (types de contrat, permis, ROME, géographie). Chaque endpoint renvoie un tableau paginé de paires (code, libelle). Tous en accès direct (partenaire, pas de jeton usager).
