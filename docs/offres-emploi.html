Accès public — Cette API est accessible en client_credentials (OAuth2, royaume /partenaire). Aucun jeton usager requis. Scopes : o2dsoffre et api_offresdemploiv2. La pagination est limitée à 150 résultats par requête (max index 3149).
Tri et pagination — Le paramètre range (ex. 0-49) pagine les résultats. Le header Content-Range indique la position dans le jeu total. Trois modes de tri : 0 pertinence · 1 date · 2 distance.
