Authentification partenaire — Cette API est dans le royaume /partenaire (client credentials). Aucun jeton usager n'est nécessaire. Les salons en ligne sont retournés sans filtre. Les événements emploi acceptent un POST avec critères de recherche (date, lieu, type, objectifs, publics).
Concepts partagés — Voir les concepts croisés entre APIs dans le catalogue
Sept taxonomies, une centaine de codes — L'API ne filtre pas uniquement sur des dates et lieux : chaque événement est tagué sur plusieurs dimensions (type, objectif, public ciblé, opération nationale, secteur d'activité). Les codes sont numériques (types, objectifs, publics, opérations) ou alphabétiques (modalité, secteur) selon la dimension. Tous sont déclarés en enum dans le schéma de requête, avec leurs libellés fournis directement par l'OpenAPI — c'est donc une doc exhaustive de ce qu'on peut passer en critère de filtre.

Secteur d'activité 14 grands domaines ROME (A–N) champ secteurActivite · string
