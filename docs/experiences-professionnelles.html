Prérequis — Cette API est dans le royaume PE Connect (OAuth utilisateur). L'usager se connecte avec son compte France Travail et autorise les scopes api_peconnect-experiencesv1 et pfcexperiences. L'application accède alors aux données de l'usager connecté uniquement.
Les données proviennent du profil de compétences saisi par l'usager lui-même sur francetravail.fr (onglet « parcours »). Elles alimentent également le système de rapprochement entre CV et offres d'emploi.
