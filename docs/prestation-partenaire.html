Base api.francetravail.io/partenaire/partenaire-gerer-prestations/v1
Qui fait quoi — France Travail prescrit les demandeurs (Candidature). Le prestataire les positionne sur des sessions, enregistre les présences, saisit les résultats et téléverse les livrables. Les commandes sont créées automatiquement lors du positionnement.
