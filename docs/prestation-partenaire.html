Concepts partagés — Voir les concepts croisés entre APIs dans le catalogue
Plus d'une trentaine de vocabulaires — Cette API a un périmètre métier dense : elle couvre le cycle de vie complet d'une prestation partenaire (candidature → session → rendez-vous → résultat) et chaque étape porte ses propres codes. Parfois le même concept est codé différemment selon le DTO (ex. la nature d'animation est I/E dans Session mais INTERNE/EXTERNE dans les filtres de recherche — même sens, encodages divergents). Quelques libellés ne sont pas documentés dans l'OpenAPI et sont marqués comme tels.
