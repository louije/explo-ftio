Lien avec les autres API ROME 4.0 — Les codes de compétences et de savoirs correspondent aux mêmes codes dans l'API ROME 4.0 Compétences. Les fiches métiers organisent ces compétences par enjeu et ces savoirs par catégorie, spécifiquement pour chaque métier.
Authentification — API publique, royaume /partenaire (client credentials). Scopes requis : nomenclatureRome + api_rome-fiches-metiersv1. Aucun jeton usager nécessaire.

📋 Fiche métier

M1805 — Études et développement informatique

Fiche ROME active du référentiel 4.0, décrivant les métiers du développement logiciel : développeur, analyste-programmeur, ingénieur d'études, etc.

🎯 Compétences mobilisées

Enjeu : Numérique

Compétences techniques liées au développement informatique :

Programmer en langage informatique → Macro savoir-faire
Développer des applications en Java → Compétence détaillée
Développer des applications en Python → Compétence détaillée

Enjeu : Organisation

Qualités comportementales attendues :

Faire preuve de rigueur et de précision → Savoir-être
Travailler en équipe → Savoir-être
📚 Savoirs

Informatique et télécommunications

Algorithme → Connaissance fondamentale
Langages de programmation informatique → Connaissance technique
Méthodes agiles → Connaissance méthodologique

Langues

Anglais technique → Connaissance linguistique

Les codes de compétences et de savoirs sont les mêmes que dans le référentiel Compétences (API rome-competences). La fiche métier les organise par enjeu (compétences) et par catégorie (savoirs) pour chaque métier spécifique.
