Authentification — API publique dans le royaume /partenaire. Accès par client credentials uniquement (scope offresetdemandesemploi). Aucun contexte utilisateur requis.
Structure commune — Tous les indicateurs (DE_1, DE_5, OFF_1, EMB_1, DYN_1, PERSP_2, SAL_3) retournent le même objet IndicateurRetour. Seuls les champs renseignés changent selon l'indicateur (nombre, taux, montant ou rang). Les référentiels (territoire, activité, période, nomenclature, caractéristique) sont partagés et consultables via des endpoints GET dédiés.
