Statut de validation email statutValidationEmail
Obligatoire string · 1:1
Indique si l’adresse email a été vérifiée par l’utilisateur
statutValidationEmail · V (validé)|P (provisoire)|I (invalide)|E (erroné)
Prérequis — Cette API est dans le royaume /individu (PE Connect). L’utilisateur doit s’être authentifié via France Travail Connect avec les scopes api_peconnect-coordonneesv1 et coordonnees. L’application accède aux données de l’utilisateur connecté — aucun header agent n’est nécessaire.
Les champs adresse1 à adresse4 suivent la norme postale française : destinataire, complément d’adresse, complément de distribution, voie. Tous les champs d’adresse sont facultatifs — seuls nom, prenom, email et statutValidationEmail sont garantis.
