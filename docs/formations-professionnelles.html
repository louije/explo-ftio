Prérequis — Cette API est dans le royaume PE Connect (OAuth utilisateur). L'usager se connecte avec son compte France Travail et autorise les scopes api_peconnect-formationsv1, pfcformations (formations) et pfcpermis (permis). L'application accède aux données de l'usager connecté uniquement.
Les niveaux de qualification suivent une échelle nationale : NV1 = Bac+5 et plus, NV5 = CAP/BEP, AFS = aucune formation scolaire. Le domaine utilise le référentiel ROME (Répertoire Opérationnel des Métiers et des Emplois).
Concepts partagés — Voir les concepts croisés entre APIs dans le catalogue
Deux référentiels documentés, beaucoup partagés — Cette API porte deux vocabulaires fermés mais sans les déclarer en enum dans les DTO. Les codes et libellés sont en revanche listés exhaustivement dans les descriptions des champs niveau.code et Permis.code. Les deux sont partagés avec d'autres APIs : le niveau d'étude avec l'API Orientation, les catégories de permis avec Offres d'emploi.

Niveau de formation 10 codes champ niveau.code — liste dans la description OpenAPI

Grille nationale française des niveaux de formation, du plus faible (aucune formation) au plus élevé (Bac+5 et plus). Même codage utilisé par l'API Orientation (champ niveau_etude) et servi par /v2/referentiel/niveauxFormations.
  • AFS Aucune formation scolaire
  • CP4 Primaire à 4ème
  • CFG 4ème achevée
  • C3A BEPC ou 3ème achevée
  • C12 2nd ou 1ère achevée
  • NV5 CAP, BEP ou équivalent
  • NV4 Bac ou équivalent
  • NV3 Bac+2 ou équivalent
  • NV2 Bac+3, Bac+4 ou équivalent
  • NV1 Bac+5 et plus ou équivalent
L'ordre naturel est présenté ici (du plus bas au plus haut). Le schéma liste les codes en désordre. Le champ etranger booléen indique si la formation a été obtenue hors de France, le diplomeObtenu si le diplôme a été validé.

Permis de conduire 20 codes champ Permis.code — liste dans la description OpenAPI

Catégories réglementaires du permis de conduire français. Regroupe tous les permis (moto, voiture, poids lourd, transport de personnes) et leurs mentions spécifiques (remorque, PTAC…). Référentiel également servi par /v2/referentiel/permis.
Deux-roues4
  • AM Cyclomoteur <= 50 cm³
  • A1 Motocyclette <= 125 cm³
  • A2 Moto <= 600 cm³
  • A Moto (toutes cylindrées)
Véhicule léger5
  • B1 Quadricycle
  • B Véhicule léger
  • B79 B mention 79.06 (= EB)
  • B96 B mention 96 — VL + remorque (PTAC < 4,25 T)
  • BE Véhicule léger + remorque (PTAC > 4,25 T)
Poids lourd4
  • C1 Poids lourd < 7,5 T
  • C Poids lourd
  • C1E Poids lourd < 7,5 T + remorque
  • CE Poids lourd + remorque (= EC)
Transport de personnes4
  • D1 Transport <= 16 places
  • D Transport de personnes
  • D1E Transport <= 16 places + remorque
  • DE Transport + remorque (= ED)
Codes d'équivalence historiques3
  • EB Véhicule léger + remorque (= B79)
  • EC Poids lourd + remorque (= CE)
  • ED Transport de personnes + remorque (= DE)
Codes doublons de l'ancienne numérotation, encore présents dans le référentiel.
