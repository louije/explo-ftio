Prérequis — Cette API est dans le royaume PE Connect (OAuth utilisateur). L'usager se connecte avec son compte France Travail et autorise les scopes api_peconnect-formationsv1, pfcformations (formations) et pfcpermis (permis). L'application accède aux données de l'usager connecté uniquement.
Les niveaux de qualification suivent une échelle nationale : NV1 = Bac+5 et plus, NV5 = CAP/BEP, AFS = aucune formation scolaire. Le domaine utilise le référentiel ROME (Répertoire Opérationnel des Métiers et des Emplois).
