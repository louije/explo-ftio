Contexte de travail ContexteTravail
Unique object · 1:1
Un élément décrivant l'environnement dans lequel s'exerce un métier : conditions physiques, horaires, lieux, statut, public, ou type de structure. Chaque contexte appartient à une catégorie et possède un code unique.
code · libelle · categorie
Conditions de travail CONDITIONS_TRAVAIL
Risques professionnels et conditions physiques d'exercice : travail en hauteur, exposition au bruit, port de charges, etc.
Horaires et durée HORAIRE_ET_DUREE_TRAVAIL
Organisation temporelle : travail de nuit, horaires décalés, temps partiel, saisonnier, etc.
Lieux et déplacements LIEU_ET_DEPLACEMENT
Où s'exerce le travail et mobilité requise : déplacements fréquents, télétravail, chantier, etc.
Statut d'emploi STATUT_EMPLOI
Forme juridique du contrat : CDI, CDD, intérim, indépendant, fonctionnaire, etc.
Publics spécifiques TYPE_BENEFICIAIRE
Types de publics concernés par le métier : enfants, personnes âgées, personnes en situation de handicap, etc.
Types de structure TYPE_STRUCTURE_ACCUEIL
Nature de l'organisme employeur : association, collectivité, entreprise privée, établissement de santé, etc.
Structure — Le référentiel est plat : chaque contexte de travail est un objet simple avec un code, un libellé et une catégorie (enum à 6 valeurs). Les catégories ci-dessus illustrent les 6 rubriques du référentiel.
Authentification — API publique, royaume /partenaire (client credentials). Scopes requis : nomenclatureRome + api_rome-contextes-travailv1. Aucun jeton usager nécessaire.

Contextes de travail — extrait du référentiel

Chaque entrée est un triplet simple : code, libellé, catégorie. Les catégories décrivent les grandes dimensions du contexte de travail.

En club sportif → CONDITIONS_TRAVAIL
Travail en hauteur → CONDITIONS_TRAVAIL
Travail de nuit → HORAIRE_ET_DUREE_TRAVAIL
📍 Lieux, statuts et structures

Autres dimensions du référentiel

Déplacements professionnels → LIEU_ET_DEPLACEMENT
Travailleur indépendant → STATUT_EMPLOI
Personnes en situation de handicap → TYPE_BENEFICIAIRE
Association → TYPE_STRUCTURE_ACCUEIL

Le référentiel complet contient plusieurs centaines de contextes répartis dans les 6 catégories. Chaque contexte peut être rattaché à une fiche métier via l'API Fiches métiers.
