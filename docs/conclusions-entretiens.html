Prérequis — Cette API est dans le royaume /agent (client credentials). Chaque appel nécessite le header ft-jeton-usager obtenu via l'API Rechercher usager, ainsi que les trois headers d'identification agent : pa-nom-agent, pa-prenom-agent, pa-identifiant-agent. Le jeton est volatile (~1 journée).
Concepts partagés — Voir les concepts croisés entre APIs dans le catalogue
Deux taxonomies, 29 codes — Chaque entretien porté par l'API est tagué sur deux dimensions : son type (21 codes pour distinguer un entretien d'indemnisation d'un bilan CRP, d'un diagnostic Cap emploi, etc.) et sa modalité de contact (8 codes pour distinguer physique, téléphone, visioconférence, courriel). Les deux listes sont déclarées en enum dans l'OpenAPI — mais les libellés tels que stockés sont tronqués (chaînes longues coupées à ~28 caractères, convention héritée du SI d'origine).

Type d'entretien 21 codes champ typeEntretien

Le type décrit la raison de l'entretien — à la fois son objet métier (diagnostic, bilan, contrôle…) et le dispositif France Travail associé (PEC/CIE, CRP, PPAE, Cap emploi, etc.).
  • ESI Entretien de situation
  • EDP Entretien personnalisé
  • EDA 1er entretien d'accompagnement
  • EDO Entretien d'orientation
  • ETP Entretien point d'étape
  • USA Entretien usager
  • IND Entretien indemnisation
  • CRL Entretien longue durée
  • CRE Contrôle recherche emploi
  • SEF Suivi emploi / formation
  • PAK Entretien pack de démarrage (PPAE)
  • EPK Entretien PPAE pack différé
  • PE1 Entretien tripartite PEC / CIE
  • PE2 Entretien de suivi PEC / CIE
  • PE3 Entretien bilatéral sortie PEC / CIE
  • PBI Pré-bilan CRP
  • CRP Entretien actualisation CRP
  • BIL Bilan CRP
  • PSP Plan de sécurisation professionnel
  • CPS Conclusion psychologue du travail
  • DCE Diagnostic croisé Cap emploi
Libellés d'origine tronqués à ~28 caractères dans le schéma (convention SI historique). Les libellés ci-dessus sont la version humainement lisible. PEC / CIE = contrat aidé · CRP = Convention de Reclassement Personnalisé · PPAE = Projet Personnalisé d'Accès à l'Emploi.

Modalité de contact 8 codes champ codeModaliteContact

Comment l'entretien s'est physiquement déroulé : en face à face, par téléphone, par visioconférence, ou par simple échange de courriel. Plusieurs déclinaisons de la visio selon l'outil utilisé côté FT (visioguichet vs visioconférence classique).
  • I Entretien individuel physique
  • R Entretien collectif physique
  • T Entretien individuel téléphonique
  • V Entretien individuel visioguichet
  • W Entretien individuel visioconférence
  • F Entretien visioconférence d'information
  • B Bilan en ligne
  • C Échange courriel
