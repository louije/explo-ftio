Ajout de compétence

Enregistrer des compétences ROME dans le portefeuille de compétences de l'usager

Compétences à enregistrer competences

Liste des compétences ROME à ajouter au portefeuille de l'usager. Chaque élément contient un code du référentiel ROME 4.0.

Compétences en erreur competencesEnErreur
0 ou plusieurs array · 0:N
Liste des compétences qui n'ont pas pu être enregistrées, avec le code ROME concerné et la raison de l'échec
codeRome · raison
Prérequis — Cette API est dans le royaume PE Connect (OAuth individu). L'usager doit s'être identifié via le bouton « Se connecter avec France Travail ». Scopes requis : portefeuillecompetencesW et api_peconnect-gestion-profilcompetencesv1.
Les codes ROME proviennent du référentiel ROME 4.0. Un code invalide ou déjà présent dans le portefeuille peut générer une réponse 206 avec le détail des erreurs. Un succès complet retourne un 204 sans corps de réponse.

POST
Enregistrer des compétences
Ajouter un lot de compétences ROME au portefeuille de l'usager connecté
/competences/enregistrer
2 Schémas — Types de données
Écriture Erreur / Utilitaire
RequestBody — Lot de compétences à enregistrer
PartialErrorResponse — Détail des compétences en erreur (206)
Enrichissement du portefeuille — Sophie, développeuse web en reconversion, utilise une application partenaire qui identifie trois compétences ROME à ajouter à son profil. L'une des compétences est déjà présente dans son portefeuille.
// POST /competences/enregistrer // Authorization: Bearer <token PE Connect> // Content-Type: application/json { "competences": [ { "codeRome": "100684" }, { "codeRome": "100712" }, { "codeRome": "100455" } ], "origine": "ORGACP" }
// → Réponse 204 No Content // Toutes les compétences ont été enregistrées // (pas de corps de réponse)
// → Réponse 206 Partial Content // Une compétence n'a pas pu être ajoutée { "competencesEnErreur": [ { "codeRome": "100455", "raison": "Compétence déjà présente dans le portefeuille" } ] }
📤 Requête d'ajout

3 compétences soumises

L'application partenaire ORGACP envoie un lot de 3 codes ROME à ajouter au portefeuille de Sophie.

100684 → Compétence en développement web
100712 → Compétence en intégration
100455 → Compétence en gestion de projet
✅ Succès complet (204)

Toutes les compétences acceptées

Le serveur retourne 204 No Content — les 3 compétences ont été ajoutées au portefeuille.

Aucun corps de réponse — l'absence de contenu signifie que tout s'est bien passé
⚠️ Succès partiel (206)

2 compétences acceptées, 1 rejetée

Le serveur retourne 206 Partial Content avec le détail des erreurs.

100455 → Déjà présente dans le portefeuille
Les compétences 100684 et 100712 ont bien été ajoutées malgré l'erreur sur la troisième
