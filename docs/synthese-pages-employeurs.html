Accès public — API accessible en client_credentials (OAuth2, royaume /partenaire). Scopes : pages-employeurs-synthese et api_synthese-pages-employeursv1. Aucun jeton usager requis.
SIREN vs SIRET — Une page employeur peut être liée soit à un SIREN (entreprise entière, avec plusieurs établissements) soit à un SIRET (établissement unique). Le champ sirenOrSiret détermine lequel est présent. Si c'est un SIREN, le champ employeur est renseigné ; si c'est un SIRET, c'est etablissement.
