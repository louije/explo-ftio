Date de naissance saisie par l’individu sur francetravail.fr, au format ISO 8601
dateDeNaissance · format YYYY-MM-DDTHH:MM:SS+HH:MM — requis
Prérequis — Cette API est dans le royaume /individu (PE Connect). L’utilisateur doit s’être authentifié via France Travail Connect avec les scopes api_peconnect-datenaissancev1 et datenaissance. L’application accède aux données de l’utilisateur connecté — aucun header agent n’est nécessaire.
Cette API fait partie du dispositif de portabilité des données personnelles. Elle retourne un seul champ — c’est l’API la plus simple du catalogue France Travail. La date inclut un fuseau horaire (généralement +01:00 pour la France).
